# GitHub Pages (Privacy Policy)

Эта папка используется для [GitHub Pages](https://pages.github.com/).

После включения Pages ссылка на политику конфиденциальности будет:

**`https://<USERNAME>.github.io/<REPO>/`**

Например: `https://myuser.github.io/aoe2/`

## Как включить

1. На GitHub открой репозиторий → **Settings** → **Pages** (в левом меню).
2. В блоке **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (или `master`), папка **/docs**
3. Сохрани (**Save**). Через 1–2 минуты страница будет доступна по ссылке выше.

Текст политики лежит в корне репозитория в файле `.privacy_polycy`; в `docs/index.html` — та же политика, оформленная для браузера.
